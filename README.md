# MindMate AI

## Description
MindMate AI is a productivity and study assistant powered by AI techniques including NLP, sentiment analysis, and optimization strategies.

## Features
AI chat assistant (NLP)
Sentiment detection (mood analysis)
AI-generated study planner
Adaptive optimization logic

## AI Techniques Used
Natural Language Processing (chat system)
Sentiment Analysis (keyword-based model)
Genetic Algorithm concept (study plan optimization simulation)

## How to Use
1. Open the app
2. Type a message (e.g. "I feel stressed about exams")
3. Receive AI feedback
4. View generated study plan

## Tech Stack
React
JavaScript
